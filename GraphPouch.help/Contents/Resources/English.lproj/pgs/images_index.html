Images
Images can be imported from third-party applications or exported from GraphPouch using copy and paste.

  • Import Image
    To import an image use one of the following methods:
    • In GraphPouch, press Control-I (^I).
    • In GraphPouch select Worksheet > New Image.
    • Press the New Image button

    A popup dialogue will appear prompting you to select an image. Select the image that you would like to import into GraphPouch and then click 'Open'. A new image object will appear on the current page.
  • Export Image
    To export an image from GraphPouch follow these steps:
    1. Select the image you would like to export by clicking on the image using the left mouse button. The selected image will have a blue rectangle designating its selected state.
    2. Select from the menu bar Edit > Copy.
    3. Switch to the application that you would like to paste the image into.
    4. Select from the menu bar Edit > Paste.
  • Remove Image
    To remove a image follow these steps:
    1. Select the image you would like to delete by clicking on the image using the left mouse button. The selected image will have a blue rectangle designating its selected state.
