Lines
Lines can be drawn on the worksheet in order to create borders, section dividers or labels.
Line
Selected line on a worksheet

  • Create Line
    To create a new line use one of the following methods:
    • In GraphPouch, press Control-N (^N).
    • In GraphPouch select Worksheet > New Line.
    • Press the New Line button
  • Remove Line
    To remove a line follow these steps:
    1. Select the line you would like to delete by clicking on the line using the left mouse button. The selected line will have a blue rectangle designating its selected state.

      Optionally you can select multiple lines by clicking on the line and pressing the Shift or âŒ˜ key.
    2. Press the Delete key.
