GraphPouch Interface
GraphPouch has one main window that displays the worksheet elements of your worksheet and one auxiliary window that allows you configure the properties of the worksheet elements.

Main Window
GraphPouch Interface contains three different components: Toolbar,  Pages Panel,  and Page View


  • Pages Panel
    The Pages Panel is a view where the user can organize and navigate their worksheet by clicking on any of the pages as well as reordering them by dragging.
  • Page View
    To modify worksheet elements the Page View is the main window for users to view each individual page. When worksheet elements are added to the page they will be displayed here.
  • Toolbar
    The Toolbar contains several buttons that you can use in order to customize your worksheet. In addition you can also bring up the Properties Panel or create a new worksheet by clicking the New Worksheet button.
Properties Panel
The Properties Panel with graph configuration
  • The Properties Panel allows the user to configure the worksheet and its elements using customized settings.
