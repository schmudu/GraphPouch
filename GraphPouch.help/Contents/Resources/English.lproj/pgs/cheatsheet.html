Cheatsheet
  • Worksheet + Pages Panel
    To use these shortcuts make sure the Pages Panel is selected. You can verify this by viewing a blue border around the edge of the Pages Panel.
    New Worksheet Command-N (⌘N)
    New Page Control-P (^P)
    Select All Pages Command-A (⌘A)
    Deselect All Pages Command-D (⌘D)
    Go to Next Page Shift-Command-] (Shift⌘])
    Go to Previous Page Shift-Command-[ (Shift⌘[)
  • Worksheet Elements
    To use these shortcuts make sure the Page View is selected. You can verify this by viewing a blue border around the edge of the Page View.

    Create Expression Control-E (^E)
    Create Graph Control-G (^G)
    Create Image Control-I (^I)
    Create Line Control-L (^L)
    Create Textbox Control-T (^T)
    Edit Textbox Double click on text
    Exit Text Edit Mode Click on Page View or press ESC key
    Select All Elements Command-A (⌘A)
    Deselect All Elements Command-D (⌘D)
    Select Next Element Alt-Command-] (⌥⌘])
    Select Previous Element Alt-Command-[ (⌥⌘[)
