Page View
The Page View is the main utility that you will use to insert, remove and modify elements from your worksheet. You can move, cut, copy, and paste worksheet elements using the Page View.

    • Equations and Expressions

  • Move Worksheet Elements
    You can move worksheet elements around by simply dragging them around the Page View.
    Note: GraphPouch has guides that enable you to line up worksheet elements in the Page View. To toggle guides press ⌥⌘G or from the Menu select Worksheet > Snap to Grid.
  • Copy Worksheet Elements
    You can copy elements in the Page View by:
    1. Click on the element you would like to copy with the left mouse button.

      The element should have a blue box around it signifying that it's selected. Optionally you can select multiple elements by clicking on several elements while holding the Shift or ⌘ key.

      Worksheet element selected
    2. Press Control-C (^C) or from the Menu select Edit > Copy.
  • Cut Worksheet Elements
    You can cut elements in the Page View by:
    1. Click on the element you would like to cut with the left mouse button.

      The element should have a blue box around it signifying that it's selected. Optionally you can select multiple elements by clicking on several elements while holding the Shift or ⌘ key.

      Worksheet element selected
    2. Press Control-X (^X) or from the Menu select Edit > Cut.
  • Paste Worksheet Elements
    You can paste elements in the Page View assuming you have already cut or copy elements using the instructions listed above. In order to paste elements:
    1. Press Control-V (^V) or from the Menu select Edit > Paste.
